## ז. מה האלגוריתם מחפש — ואיך UPE מכסה את זה

מנהל Instagram, Adam Mosseri, אישר שלושה אותות דירוג מרכזיים: Watch time (35% משקל) — כמה זמן אנשים צופים בתוכן שלך. Completion rate חשוב יותר מ-views. Reel של 30 שניות עם 80% completion עוקף Reel של שתי דקות עם 30% completion.

עבור Reels, שליחה דרך DM היא האות ממושקל ביותר להפצה.

| אות אלגוריתמי | איך "אירוע אחד, 60 שניות" מכסה אותו |
|---|---|
| **Completion rate** | 45–60 שניות בלבד → קל לסיים; מבנה תמציתי עם payoff בסוף |
| **Shares/DMs** | תוכן BTS "בלתי נראה" → שווה שיתוף לעמית/מנהל |
| **Saves** | מידע מקצועי (איך מנהלים crisis) → שמירה לצורך עתידי |
| **Replays** | רגעי wow ויזואליים → צופים שוב |
| **Contextual continuity** | מילות מפתח זהות ב-audio + subtitle + caption |

העקרון המוביל ב-2026 הוא Contextual Continuity — וידוא שהאודיו, הטקסט על המסך, וה-caption משתמשים באותן מילות מפתח סמנטיות. האלגוריתם של Instagram כיום "מאזין" לסרטון שלכם, וכשהאלמנטים מתואמים, ה-Search Discoverability עולה.

**מילות המפתח הסמנטיות לשימוש עקבי ב-UPE Reels:**
`event production` / `הפקת אירועים` / `corporate event` / `incentive travel` / `אירועי חברה` / `conference production` / `ועידות` / `team building`

## ח. Workflow פרודקשן — איך מייצרים 2 Reels בשבוע בלי לשרוף את הצוות

Batch production עובד. מקליטים כמה קליפים בסשן אחד. מייעדים אדם אחד בצוות לצלם BTS תוך כדי שהאירוע קורה. המתנה לייצר אותו אחרי האירוע מפספסת את המטרה.

```
WORKFLOW — כל אירוע שUPE מפיק:

[DURING EVENT]
 → 1 איש צוות (או מצלמן חיצוני) עם iPhone
 → 15–20 קטעים גולמיים × 30–90 שניות כ"א
 → לפחות: hook shot / crisis moment / payoff shot / reaction

[POST EVENT — תוך 48 שעות]
 → עורך מרכיב 2 גרסאות מהחומר
 → Subtitle עברי + אנגלי נוסף
 → Review + אישור Alon (Draft)
 → Schedule לשלישי/חמישי הקרובים

[BATCH]
 → מכל אירוע ניתן לשלוף 2–4 Reels
 → חודש אחד של אירועים = 2–3 חודשים של תוכן
```

צמיחת Instagram Reels ב-2026 מתכנסת לשני דברים: איכות ועקביות. לא צריך ציוד מקצועי. צריך אסטרטגיה ברורה, מערכות חוזרות, וכלים נכונים.
