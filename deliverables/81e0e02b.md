## EXECUTION PLAYBOOK — כיצד מבצעים זאת

### 📅 שבועי Workflow (Batch Production)

גישת ה-batch creation: 2–3 שעות פעם בשבוע — צלמו 5–7 סרטונים קצרים בסשן אחד (15–60 שניות כל אחד), תאורה טבעית, מצלמת טלפון, רקע פשוט. Script בנקודות בלבד — לא מילה במילה.

```
יום שני:   PLANNING — בחרו את האירוע הקרוב, הגדירו את ה-hook
יום שלישי: SHOOT DAY — במקום האירוע עצמו (BTS אמיתי)
יום רביעי: EDIT — כתוביות, text overlays, trim
יום חמישי: 2–3 סרטונים לאוויר
יום שישי:  1–2 סרטונים נוספים + תגובות לקומנטים
```

### 🎣 Hook Formula Sheet (לשמור ולהדביק)

ה-hook המוביל בביצועים הוא Product/Outcome Showcase — פתיחה עם התוצאה, הטרנספורמציה, או המספר בשתי השניות הראשונות. ממוצע 6,037 צפיות לקליפ — הגבוה מכל סוג hook.

| Hook Type | נוסחה | דוגמה לUPE |
|---|---|---|
| **Numbers First** | [מספר] + [פעולה] + [זמן] | "48 שעות לפני כנס של 500 איש" |
| **Problem Hook** | [בעיה] + [זמן לפתרון] | "הבמה הגיעה שבורה. שעה אחת." |
| **Outcome First** | [תוצאה] ← [כיצד?] | "0 בעיות שהם ידעו עליהן. ככה." |
| **Curiosity Gap** | שאלה שהתשובה לה מפתיעה | "120 אנשים לכנס של 1,000. למה?" |
| **Storytime** | "זה קרה לנו ממש…" | "הספק לא הגיע ב-D-1. ככה יצאנו ממנו." |

Curiosity Gap hooks יוצרים עניין בתוך 3 השניות הראשונות. Controversy ו-Bold Claims בולטים בפיד עמוס. Question hooks מניעים engagement דרך reflection.

### ⚠️ מה לעשות — ומה לא

פתיחות שלא עובדות: "Let me tell you about the time…" (דורש אמון שעדיין לא ניתן), "Hi everyone, today we're going to talk about…" (פורמט יוטיוב — לא TikTok), ו-"Okay so…" (מבזבז את 2 השניות הראשונות).

| ✅ עשו | ❌ אל תעשו |
|---|---|
| פתחו עם מספר/תוצאה בפריים הראשון | פתיחה עם "היי כולם, היום נדבר על…" |
| כתוביות על כל סרטון | לסמוך על שמע בלבד |
| שאלה בקפשן | לבקש לייק ("כפרו" — ב-60% פחות לייקים) |
| Batch צילום ביום אירוע | לצלם מהמשרד בלי במה/ציוד |
| לענות לכל תגובה ב-24 שעות | לפרסם ולהיעלם |

קפשן עם שאלה = 26% יותר תגובות. בקשה לתגובה מעלה תגובות ב-14%, אבל בקשה ל-like — מוחקת לייקים ב-60%.

## KPIs ומה למדוד
